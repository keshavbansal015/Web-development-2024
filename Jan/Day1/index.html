Basic Structure of an HTML Document

An HTML document has a basic structure that includes doctype, html, head, and body tags. But obviously now there are so many more annoying tags.

Headings

HTML offers six levels of headings, from h1 to h6. h1 is used for main headings, followed by h2, and so on, to h6. We are not limited to these font sizes, we can adjust them with very fine control now.
