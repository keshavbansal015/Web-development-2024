Markup Languages

  • Purpose: Structure and annotate text in documents.
  • Tags and Syntax: Use of tags (like <tag> in HTML) to denote structure and formatting.
  • HTML (HyperText Markup Language): For creating web pages.
  • XML (eXtensible Markup Language): For data storage and transport.
  • Markdown: For simple text formatting.
  • Separation of Content and Presentation: Allows flexibility in how content is displayed.
  • Portability: Text-based, easily edited with any text editor.
  • Versatile Use: Across web development, data handling, and document formatting.
